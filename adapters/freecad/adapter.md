# FreeCAD Adapter

Use FreeCAD when an open-source CAD application is preferred for visualization, BREP import/export, or additional CAD healing.

Typical route:

1. Generate profile JSON with core scripts.
2. Generate STEP through CadQuery/OCC or FreeCAD scripting.
3. Open or import STEP in FreeCAD.
4. Validate shape and export desired formats.

This adapter is intentionally documentation-first until a specific FreeCAD install path and target workflow are known.
