# M1009 Asset Manifest

- `input/M1009_curved_face_block_reference.STL`: reference mesh used for sampling and fitting.
- `profiles/native_loft_profiles.json`: fitted section profile data.
- `outputs/M1009_curved_face_block_single_solid.step`: single BREP/STEP generated from fitted profiles.
- `outputs/M1009_curved_face_block_single_solid_preview.stl`: mesh preview exported from the single BREP solid.
- `outputs/M1009_curved_face_block_complete_single_solid.SLDPRT`: optional SolidWorks output from a local Windows/SolidWorks run.
- `outputs/M1009_curved_face_block_complete_single_solid_preview.png`: SolidWorks visual check image.
- `case-code/`: case-specific source code preserved for teaching.

Quality label: Q4 for the local SolidWorks deliverable, Q2 for the tool-agnostic STEP deliverable.
